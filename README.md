# 📊 Global Superstore Sales Analysis Project

## 🚀 Overview
This project analyzes a global retail dataset to extract business insights related to sales, profit, discounts, and regional performance using Python and Excel.

The goal is to demonstrate **data analysis, cleaning, visualization, and dashboarding skills**.

- Converted incorrect data types (Sales, Profit, Quantity)
- Standardized column names
- Handled missing values
- Cleaned text fields using string operations
- Created new features:
  - Profit Margin
  - Year
  - Month

## 📊 Key Business Insights

### 💰 Sales Performance
- Technology is the highest revenue-generating category.
- Furniture shows high sales but lower efficiency.

### 📉 Profitability
- Office Supplies has the highest profit margin.
- Furniture has low profitability despite high sales.

### 🧾 Discount Impact
- High discounts significantly reduce profit.
- Discounts above 0.3 often result in losses.

### 🌍 Regional Performance
- Central region leads in total sales.
- Canada has the lowest contribution.

### 📦 Product Analysis
- A small group of products contributes majority of revenue (Pareto principle).

### 📈 Trend Analysis
- Sales show steady growth from 2011 to 2014.
